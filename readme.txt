# Weather App

A simple and user-friendly weather application that displays current weather data for cities in Germany.

## Features

- Display current weather data for any city in Germany
- Search for cities in Germany
- Show temperature, wind speed, humidity, air pressure, and more
- Weather icons for visual representation of current weather conditions

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- [Brightsky API](https://brightsky.dev/) for weather data
- [OpenStreetMap Nominatim API](https://nominatim.org/) for geolocation

## Installation

1. Clone the repository:

git clone https://github.com/YourUsername/weather-app.git

2. Navigate to the project directory:

cd weather-app

3. Open `index.html` in your browser or use vite.

## Usage

1. Enter a city name in the search field.
2. Click the search button or press Enter.
3. The current weather data for the searched city will be displayed.
